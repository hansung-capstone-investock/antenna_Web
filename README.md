## Investock 팀 정세계

### Investock의 Antenna 기능을 포함한 다양한 서비스를 웹으로 제공하기 위해, REACT를 이용하였습니다.
### 기본적인 JS의 사용부터, REACT의 특징인 VirtualDOM & Component 형태를 사용하여 어플리케이션의 성능 향상을 목표로 개발하였습니다.
### 이로인해, 차트와 리스트 등 데이터 시각화에 초점을 맞추어 렌더링 시간을 최소화 하였습니다.
